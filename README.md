# Bali Komodo WeChat Itinerary

Static H5 itinerary page for WeChat in-app browsing.

Files:

- `index.html`: default publishable version with names anonymized.
- `open.html`: WeChat pinned-message entry page with a second confirmation tap.
- `index.private.html`: private preview version preserving original names.

Recommended GitHub Pages setup:

1. Create a new GitHub repository, for example `bali-komodo-itinerary`.
2. Push this folder to the repository.
3. Enable Pages from the repository root on the `main` branch.
4. Share the generated Pages URL in WeChat.
5. If WeChat pinned messages do not open the direct link reliably, pin `/open.html` instead.

Privacy note: GitHub Pages URLs are effectively public unless access is controlled by the GitHub plan and repository settings. Keep hotel names, order numbers, phone numbers, and exact private contacts out of `index.html`.
